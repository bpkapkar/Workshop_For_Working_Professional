# Workshop_For_Working_Professional

This repository is part of the Workshop_For_Working_Professional.
Follow the steps below to set up the required API keys before starting the workshop.

## Prerequisites

You will need API keys from the following platforms.

### Google AI Studio
Create your API key from Google AI Studio:
https://aistudio.google.com/api-keys

### Tavily Playground
Generate your Tavily API key from the Tavily dashboard:
https://app.tavily.com/home

### Groq
Create your Groq API key from the Groq console:
https://console.groq.com/keys

## Notes
- Do not commit API keys directly to the repository.
- Use environment variables or a .env file to manage secrets locally.
- Ensure all keys are created before starting the workshop.

## Getting Started
Once all API keys are configured, proceed with the workshop exercises.
If you want this adjusted for a specific tech stack (Python, Node.js, LangChain, FastAPI, etc.) or want environment variable examples added, let me know.
